# Avaota-A1

语言：[English](README.md) | 中文

Avaota-A1 - 一款基于全志 T527 八核 A55 SoC 的高性能单板计算机。

![IMG_1742 - 副本](https://github.com/YuzukiHD/Avaota-A1/assets/12003087/1e4e7bca-a16f-469d-96ad-aa21b29acfbf)

## 特性

- CPU：八核 Cortex-A55，最高 2.0GHz
- GPU：Mali G57，支持 OpenGL ES 3.2/2.0/1.1、Vulkan 1.1/1.2/1.3、OpenCL 2.2
- RCPU：玄铁 E906 RISC-V @ 200MHz
- DSP：HIFI4 DSP @ 600MHz
- NPU：最高 2TOPS 算力
- 内存芯片：支持 LPDDR4/4X，可选 1/2/4GB
- 存储芯片：支持 eMMC 5.1，可选 16/32/64/128GB
- 摄像头：MIPI CSI 接口，支持 1300 万像素，4-lane MIPI CSI
- 网络通信接口：2 x 千兆以太网，WIFI6 2.4G + 5G，BT5.4
- USB 接口：包括 USB OTG、USB3.0 OTG、USB2.0 HOST
- 音频接口：3.5mm 音频接口，支持录音功能
- 显示接口：包括 RGB 接口、MIPI DSI 接口、DP 接口、HDMI2.0 接口、LVDS 接口、MIPI DBI 接口
- 其他接口模块：包括调试串口、TF 卡槽、CAN 接口、RGB LED、IR、UART、IIC、I2S、SPI、PWM
